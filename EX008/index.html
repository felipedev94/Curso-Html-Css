Principais formatações

Negrito

Nessa frase temos um exemplo em Negrito usando tag <b> (Sem semântica).

Nessa frase temos outro exemplo em Destaque usando a tag <strong> (Com semântica).

Itálico/Ênfase

Nestra frase temos um termo em Italico usando a tag <i> (Sem semântica).

Nesta frase temos um termo em Ênfase usando a tag <em> (Com semântica).

Texto marcado

Podemos marcar um texto com a tag <mark> a cor foi modificada usando a tag <style>.
A cor não modificada é em aramelo.

Texto grande e pequeno

Podemos criar textos grandes utilizando a tag <big>
A tag <big> esta obsoleta por isso quando quiser aumentar determinada frase ou palavra devemos utilizar a formatação do texto.

Podemos criar textos pequenos utilizando a tag <small>.

Texto deletado

Podemos marcar textos como deletados com a tag <del>.

Texto inserido

Podemos marcar um texto como inserido (sublinhado) com a tag <ins> para da ênfase que foi colocado depois (Tag semântica).

Podemos marcar um texto como inserido (sublinhado) com a tag <u> para da ênfase que foi colocado depois (Tag não semântica).

Texto sobrescrito

Para inserir textos tipo: x20+3 utilizamos a tag <sup>

Texto subscrito

Para inserir textos tipo H2O utilizamos a tag <sub>
