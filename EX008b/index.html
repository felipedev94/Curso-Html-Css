Outras formatações

Código fonte/Pré-formatação

A tag <code> é usada para citar partes de codigos, exemplo: document.getElementByid('teste') sem a tag <code>
document.getElementByid('teste') com a tag <code>

Se a citação for de mais de uma linha onde precise de uma formatação melhor usa-se a tag <pre> exemplo de comando sem a tag :
num= int(inoput('Digite um número')) if num % 2 == 0; print (f'O número {num} é PAR') else: Print(f'O número {num} é IMPAR') print ('Fim do programa')

Exemplo do mesmo comando com a tag <pre>: 

num= int(inoput('Digite um número'))
if num % 2 == 0;
    print (f'O número {num} é PAR')
else:
    Print(f'O número {num} é IMPAR')
print ('Fim do programa')

Citações simples

Caso seja preciso usar uma citação simples existe a tag <q> exemplo:
Como diria o pai do amigo do Guanabara: O computador é um burro muito rápido

Citações completas

Para ciações mais completas usa-se a tag <blockquote> a citação usa o link da referencia, porém esse link não aparece para o usuario, serve apenas para o navegador saber onde fica a citação, exemplo:
Como esta escrito na biblia em 1° Samuel 8:7-9:

7E o Senhor disse a Samuel: Atenda à voz do povo em tudo o que lhe pedem. Porque não foi a você que rejeitaram, mas a mim, para que eu não reine sobre eles.8Segundo todas as obras que fizeram desde o dia em que os tirei do Egito até hoje, pois me deixaram e serviram outros deuses, assim também estão fazendo com você. 9Agora, pois, atenda à voz deles, porém advirta-os solenemente e explique-lhes qual será o direito do rei que vier a reinar sobre eles.

Abreviações

Para mostrar o significado de abreviações citadas usa-se a tag <abbr> exemplo:

"Estou estudando HTML e CSS"

Note que HTML e CSS é explicado deixando o mouse por cima das palavras

Texto invertido (não sei onde vou usar isso um dia)

Para inverter um texto usa-se a tag <bdo>
A frase: "O rato roeu a roupa do rei de Roma" fica: O rato roeu a roupa do rei de Roma
